# Decisions

## Versioning
- 앱 버전은 수동 문자열 대신 자동 생성 방식을 유지한다.
- 이유: Vercel/로컬 빌드 사이 불일치와 `v0` 회귀를 막기 위함.

## Telegram Host
- 기본 텔레그램 호스트는 Mac이다.
- 이유: 항상 켜두는 운영 머신을 하나로 정리해야 polling 충돌을 피할 수 있다.

## Home Ordering
- 습관 순서 변경은 드래그 방식으로 유지한다.
- 습관 드래그는 오버레이보다 원본 행 이동 방식을 우선한다.
- 이유: 모바일 터치에서 손가락 위치와 시각적 위치가 더 잘 맞는다.

## Update UX
- 설치형 앱에서는 업데이트 준비 상태를 앱 안 배너로 명시한다.
- 이유: 서비스워커 교체가 사용자가 보기엔 불투명하기 때문.
